# 🛍️ Customer Shopping Behavior Analytics

An end-to-end retail analytics project analyzing customer purchasing patterns using Python, SQL, and Power BI — translating raw transaction data into actionable business intelligence across customer segmentation, purchase drivers, and revenue trends.

---

## 📌 Project Overview

Retail businesses generate massive volumes of transaction data that often go underutilized. This project simulates a corporate-grade analytics workflow — cleaning and modeling raw customer data, running structured SQL analysis, and delivering an interactive Power BI dashboard with a final business report.

**Business Problem:** The retail team lacked visibility into customer behavior, purchase frequency, and product preferences — making it difficult to drive targeted marketing, optimize inventory, and improve revenue performance.

## ⚙️ Project Phases

**Phase 1 — Data Preparation & EDA (Python)**
- Imported and explored raw customer transaction dataset
- Cleaned and transformed data — handled nulls, standardized formats, removed duplicates
- Conducted exploratory data analysis to identify patterns before SQL loading

**Phase 2 — Data Analysis (SQL)**
- Loaded cleaned data into SQL database via Python
- Answered 15+ business questions covering:
  - Customer segmentation and loyalty patterns
  - Purchase frequency and average order value
  - Product category preferences and revenue drivers
  - Seasonal and demographic buying trends

**Phase 3 — Visualization & Insights (Power BI)**
- Connected SQL database to Power BI
- Built an interactive dashboard highlighting key customer behavior patterns
- Enabled stakeholders to slice data by segment, category, and time period

**Phase 4 — Report & Recommendations**
- Wrote a structured business report summarizing findings
- Translated data insights into actionable product strategy and marketing recommendations

---

## 🔑 Key Business Questions Answered

- Which customer segments drive the highest revenue?
- What are the most purchased product categories by age group and gender?
- How does purchase frequency vary across loyalty tiers?
- Which seasons and time periods see peak buying activity?
- What are the top revenue-driving products and underperforming categories?
